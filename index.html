Ergonomic Workstation Setup

Monitor Position

  • ✓ Top of screen at eye level
  • ✓ 20–28 inches away from eyes
  • ✓ Slight downward angle (15–20°)
  • ✓ Reduce glare with anti-glare screen

Seating

  • ✓ Feet flat on floor (90° angle)
  • ✓ Hips and knees at 90°
  • ✓ Lower back supported
  • ✓ Armrests at elbow height

Keyboard & Mouse

  • ✓ Elbows at 90° angle
  • ✓ Wrists straight, not bent
  • ✓ Mouse close to keyboard
  • ✓ Use keyboard tray if needed

Head & Neck

  • ✓ Chin parallel to ground
  • ✓ Shoulders relaxed
  • ✓ Avoid forward head posture
  • ✓ Neck in neutral position
